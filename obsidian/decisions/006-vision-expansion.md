# Expand Iris from Voice-Only to Multimodal Clinical Intelligence Platform
Date: 2026-03-06
Made By: Waiz Khan
Reason: Research shows multimodal sensing (voice biomarkers + photo-based edema detection + structured vitals) dramatically outperforms any single modality for predicting heart failure decompensation. Home foot-scanning predicted hospitalizations 13 days in advance — better than daily weight alone. Voice biomarkers discriminate wet/dry states with 98% accuracy. Nobody is combining all three sensing modalities in a patient-facing autonomous agent. This is a genuine competitive advantage that aligns with what ARPA-H ADVOCATE is funding.
Impact: Architecture expands from "voice agent with clinical tools" to "multimodal clinical intelligence platform." New capabilities needed: voice biomarker extraction during conversations, image analysis for edema/medication/weight photos, behavioral state machine (COM-B + Transtheoretical Model). Patient state model becomes 4-layer (clinical, behavioral, communication, social). SOUL.md rewritten to articulate the expanded vision. CLAUDE.md updated with new technical direction and clinical roadmap. No existing code changes — these are vision/documentation updates that set the direction for implementation.
